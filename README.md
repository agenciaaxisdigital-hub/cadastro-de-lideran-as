# Cadastro de Lideranças — Dra. Fernanda Sarelli

Aplicação PWA para gestão de lideranças de campanha.

## Tecnologias

- React 18 + TypeScript
- Vite + Tailwind CSS + shadcn/ui
- Supabase (auth + banco de dados)
- React Query + React Hook Form + Zod

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse em `http://localhost:8080`.

## Build

```bash
npm run build
```
